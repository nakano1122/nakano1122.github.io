ユーザーの嗜好や検索コンテキストに応じて、より関連性の高い検索結果を提供するためのパーソナライゼーション手法を研究しています。

従来の検索システムでは、同じクエリに対して全てのユーザーに同一の結果を返しますが、ユーザーごとに求める情報は異なります。本研究では、ユーザーの過去の行動履歴や現在のコンテキストを活用し、個々のニーズに合わせた検索結果のランキングを実現することを目指しています。

<!-- lang:en -->

We are researching personalization methods to provide more relevant search results based on user preferences and search context.

Traditional search systems return the same results for all users given the same query, but each user seeks different information. This research aims to leverage users' past behavior history and current context to achieve search result ranking tailored to individual needs.

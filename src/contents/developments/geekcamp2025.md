チームでのソフトウェア開発の実践として、すれ違ったユーザのアカウントを検知したり、QR コードで共有可能な iOS アプリを開発しました。
開発は 4 人で行い、私は、すれ違ったユーザを取得する API、フレンドを追加する API などのバックエンドの開発を担当しました。

<!-- lang:en -->

As a team software development project, we developed an iOS app that detects nearby users and enables account sharing via QR code.
Our team of 4 built this project, and I was responsible for backend development including the nearby user detection API and friend request API.

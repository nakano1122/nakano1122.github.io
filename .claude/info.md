# 掲載する情報

## TOP 画面

### ヒーローセクション

- 氏名
- 所属
  - 大学
  - 学部・学科
  - 研究室（研究室の公式 Web サイトのリンクをつけます）
- 連絡先
  - メール
  - GitHub

### 基本情報

- 氏名
- 所属
- 学年
- TOEIC
- 資格

### アプリ開発経験

- ハッカソン受賞歴
- 技術スタック（ロゴで表示します）
- 個人開発プロジェクト（タイトルのみ）

### 研究内容

- 研究キーワード
- プロジェクト
- 発表済み論文
  - 論文タイトル
  - 著者

### インターンシップ歴

- 会社名（日付が新しいものを上に表示します。）
- 期間
- 役職

## アプリ開発経験ページ

- 技術系イベントでの受賞歴全て
  - イベント名
  - 結果（最優秀賞など）
  - 日付
    - 西暦年
    - 月（month）

## 研究内容ページ

- 研究内容のテキスト
- 発表済み論文
  - 発表会議
  - 論文タイトル
  - 著者
  - アブストラクト

## インターンシップ歴ページ

終了日時が新しいものを上位に表示してください。なお、終了日時には「現在」という項目も含みます。

- 会社名
- 期間
- 業務内容（テキスト）
- 業務で使用した技術スタック（ロゴ）
